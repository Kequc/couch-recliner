CouchOperations
===

This tool uses a [Couch](./couch) instance, to perform low level database operations.

### uuids

Retrieve ids generated by CouchDB.

```javascript
const { CouchOperations } = require('couch-recliner');

CouchOperations.uuids(myCouch, 5, (err, ids) => {
    if (!err)
        console.log(ids);
});
```
