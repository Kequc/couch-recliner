CouchOperations
===

```javascript
const { CouchOperations } = require('couch-recliner');
```

### .nextId(callback)

| parameter | description |
| - | - |
| callback(err, id) | Returns an id generated by the database. |

```javascript
CouchOperations.nextId(couch, (err, id) => {
    if (!err)
        console.log(id);
});
```
